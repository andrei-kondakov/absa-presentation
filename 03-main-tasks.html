Основные задачи

  • Разработка системы с использованием алгоритмов машинного обучения для извлечения аспектов и тональности предложений в отзывах о ресторанах
  • Использование в качестве размеченных выборок для обучения и тестирования данные с соревнования Semeval 2016
  • Сравнение работы системы c результатами участников соревнования
  • Реализация платформы для автоматизации исследований
  • Реализация программы для демонстрации работы системы
Постановка задачи анализа тональности разнообразна: определение тональности всего текста, отдельных предложений, конкретных фраз и терминов. Более формально проблема анализа тональности называется аспектно-эмоциональным анализом, где аспект – это интересующая сущность и его признак или категория.
